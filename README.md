# Dreamery

Save your dreams forever to relive through the dream journal. It creates videos so you can relive the full experience of the dream again.

## Features

- Add your dream to the dream archive
- Choose a video of 5, 10 or 15 seconds
- Search and filter your dreams in the archive to relive past dream memories
- Dark mode support

## Tech Stack
- HTML
- CSS
- JavaScript
- Vercel
- MongoDB
- MagicHour API
